## 安装

```shell
git clone 项目地址
cd 项目地址
npm link
```

## 使用
```shell
chan init
```
按照提示完成项目搭建

## 功能
  
- 完成交互式创建模板项目

- 完成git半自动打标签、提交、发布功能

- 完成npm自动化发布
